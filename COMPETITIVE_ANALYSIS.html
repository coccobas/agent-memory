Executive Summary

Agent Memory is a well-architected MCP server providing structured memory for AI agents with strong fundamentals: hierarchical scoping, version history, semantic search, and MDAP support. Compared to commercial competitors (Mem0, LangGraph, Anthropic Memory, Zep, Microsoft Foundry), Agent Memory has competitive core features but lacks enterprise capabilities, managed infrastructure, and go-to-market positioning.

Key Findings:

  • Strengths: Advanced MDAP support, version history, conflict detection, hierarchical scoping
  • Gaps: Enterprise features (SSO, SLA, on-prem), managed hosting, pricing model, marketing
  • Opportunity: Position as open-source alternative with enterprise add-ons, focus on MDAP/developer-first market
