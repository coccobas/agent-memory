# Contributing

Thanks for contributing to Agent Memory.

## Setup

```bash
npm install
npm run build
```

## Quality Gates

Before opening a PR:

```bash
npm run validate
```

## Style

- TypeScript strict mode is enforced.
- Follow existing patterns in `src/services` and `src/mcp`.
- Add tests for new behavior.

## Security

If you find a vulnerability, please report it privately.
