Agent Memory

Persistent, scoped memory for AI coding agents via MCP

The Problem

AI coding agents forget everything between sessions. They re-learn your codebase conventions, repeat mistakes, and lose context about decisions made yesterday.

The Solution

Agent Memory gives your AI agents a persistent brain. Store tools, guidelines, and knowledge at any scope level—from global defaults down to session-specific context. Agents remember what works, avoid what doesn't, and build on previous decisions.

Key Features

  • Hierarchical Scopes — Global → Org → Project → Session inheritance
  • Version History — Track changes with full audit trail
  • Cross-Reference Query — Find related context with relevance scoring
  • File Locks — Multi-agent coordination without conflicts
  • Tags & Relations — Rich metadata for intelligent retrieval
  • MCP Native — Works with Claude Desktop, Cursor, and any MCP client
